# Work Division Plan

## Team Roles

### Frontend Developer (1 person)
- Streamlit UI implementation
- Page routing and navigation
- File upload component
- Note display components
- Search interface, Settings UI, System status dashboard

### AI / ML Engineer (1-2 persons)
- Ollama integration
- llama.cpp integration
- Prompt engineering
- Model selection and quantization
- JSON schema validation
- Caching, graceful degradation

### Database Engineer (1 person)
- SQLite schema design
- SQLAlchemy ORM setup
- FTS5 full-text search
- CRUD operations
- Data export

### Testing / QA Engineer (1 person)
- Unit tests for all modules
- Integration tests
- Test fixtures, CI pipeline
- Coverage reporting
- Edge case testing

### Documentation (1 person)
- README, API docs, User manual
- Troubleshooting guide
- Architecture documentation
